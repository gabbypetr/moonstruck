# Moon Phase Visualizer

Moon Phase Visualizer is a Go console application that estimates the Moon phase for any given date.

The project demonstrates date calculations, modular architecture and a simple ASCII visualization.

## Features

- Calculate moon phases
- ASCII moon icons
- Age of the moon
- Sample date collection
- Statistics

## Example

```
Moon Phase Visualizer

Date

2026-07-22

Moon Age

27.8 days

Phase

Waning Crescent

Visualization

🌘

Processed dates: 5
```

Run

```bash
go run .
```
